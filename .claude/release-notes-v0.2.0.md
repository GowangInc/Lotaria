# Lotaria v0.2.0 - Custom Moods & Enhanced Pets

## 🎨 Major Features

### Custom Mood System
- Create your own personality prompts
- **AI-powered improvement**: Click "✨ Improve with AI" to enhance your custom mood using the selected vision model
- Full control over how Lotaria interacts with you

### Click-Through Window
- Transparent areas now click-through to your desktop
- Only the pet and UI elements are interactive
- No more blocking your workflow!

### Avatar Animations
- Smooth collapse animation before screenshot
- Graceful expand animation after capture
- Pet remains visible during analysis with "thinking" indicator

## ✨ Pet Visual Overhaul

All 10 pet designs have been massively improved with unique personalities:

- **📺 Retro Computer** - CRT scanlines, pulsing power LED, screen flicker
- **🐱 Cat** - Animated whiskers, ear twitches, slit pupils, breathing
- **👻 Ghost** - Ethereal glow, flowing energy trail, translucent form
- **🤖 Robot** - Metallic panels with rivets, scanning LED eyes, beeping antenna
- **🫧 Blob** - Gooey dripping effects, color-shifting gradients, jiggling motion
- **🦉 Owl** - Feather texture, independent ear tufts, wise blinking, hooting
- **👽 Alien** - Pulsing energy field, scanning sensor, otherworldly glow
- **🎃 Pumpkin** - Carved ribs, inner candle glow, flickering flame
- **☁️ Cloud** - Fluffy layered puffs, rain drops, lightning flash
- **🐙 Octopus** - Wavy tentacles with suction cups, water ripples, swimming

Each pet now features:
- Multi-layered CSS effects
- Character-specific animations
- Unique color palettes
- Environmental details (particles, glows, weather)

## 🛠️ Technical Improvements

- Added `custom_mood` field to config
- New `improve_mood` Tauri command
- Pointer-events system for click-through
- Multi-stage CSS animations
- Enhanced visual effects (gradients, shadows, filters)

## 📦 Installation

Download the MSI installer below and run it. Lotaria will install and be ready to roast!

## 🔑 Setup

1. Get a free API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Launch Lotaria
3. Enter your API key in the welcome screen
4. Start getting roasted!
